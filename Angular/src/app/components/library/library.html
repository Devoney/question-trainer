What is this app?

An effective way to learn new things is by starting with a question, find the answer, and repeat that. So when you are studying from books for instance, you can turn everything you didn't know into questions. You then find the answer for it. When you repeat the questions till you know them, you are effectively studying. This app helps you doint that.

How to use it?

  • Start by adding books.
  • For each book you then add chapters.
  • For each chapter you then add questions.
  • Put questions you want to study in the list.
  • Press start to train the questions.

How do I know I am ready for the exam?

  • When you have been honest with yourself, and you have created a question for every little thing you did not know yet.
  • You have repeated the list of questions several times while answering all questions correctly.
